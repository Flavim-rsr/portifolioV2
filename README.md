# Portfolio

Portfolio pessoal construido com Next.js, focado em apresentar projetos, experiencia e formas de contato.

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Como rodar localmente

```bash
npm install
npm run dev
```

Abra http://localhost:3000 no navegador.

## Estrutura do projeto

- Pagina principal: src/app/page.tsx
- Layout global: src/app/layout.tsx
- Secoes: src/components/sections
- Navbar e Footer: src/components
- Dados e textos: src/lib

## Onde editar

- Texto do Hero e demais secoes: src/lib/translations.ts
- Lista de skills/tecnologias: src/lib/data.ts
- Componentes visuais: src/components/sections
